Notification Center et Mountain Lion!


Si vous êtes dans mon cas : pas d'argent pour se payer un compte développeur pour avoir la doc des API de Mountain Lion en avance voici une des API majeur de ce dernier : le Notification Center (aka NSUserNotification.h)! On peut retrouver son header dans la framework Foundation sous le nom de NSUserNotification.h Mais attention, dans la DP1 il faut obligatoirement un code signé (avec le programme de dev) mais plus dans la DP2 (DP3 je ne sais pas) 
NSUserNotificationCenter *notifCenter = [NSUserNotificationCenter defaultUserNotificationCenter]; //centre de notification
NSUserNotification *notification = [[NSUserNotification alloc] init]; //on initalise notification
[notification setTitle:@"Un titre de notification!"];
[notification setInformativeText:@"Et du texte!"];
NSDictionary *infoDict = [NSDictionary dictionaryWithObject:@"test" forKey:@"nomNotif"]; //un dict pour reconnaitre la notification
[notification setUserInfo:infoDict]; //et on lui donne notre dict.
[center deliverNotification:notification];
Biensur il faut supprimer la notification et éxécuter une action lorsqu'elle est publier! ça on le fait via son delegate: 
- (void)userNotificationCenter:(NSUserNotificationCenter *)notifCenter didActivateNotification:(NSUserNotification *)notification {
	NSLog([[notification userInfo] valueForKey:@"nomNotif"]); //log = test
															//ça peut être utile pour identifier la notification
	[center removeDeliveredNotification:notification]; //on supprime la notification
}
